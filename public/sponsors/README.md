# Sponsor Logos Directory

## Place your sponsor logo files here

### Sponsor Categories & Logo Sizes:

1. **title-sponsor.png** - 800x400px (Premium placement)
2. **co-sponsor.png** - 600x300px
3. **food-sponsor.png** - 500x250px
4. **hospitality-sponsor.png** - 500x250px
5. **presentation-sponsor.png** - 500x250px
6. **water-bottle-sponsor.png** - 500x250px
7. **media-partner.png** - 500x250px
8. **entry-gate-sponsor.png** - 500x250px
9. **event-partners.png** - 400x200px

### File Format Guidelines:

✅ **Recommended:**
- PNG with transparent background
- SVG for perfect scaling
- High resolution (2x for retina displays)

❌ **Avoid:**
- JPG (unless absolutely necessary)
- Low resolution images
- Images with white/colored backgrounds (use transparent)

### Quick Setup:

Once you place your files here, they'll automatically appear on:
- Homepage
- Stats page
- Quizzes page
- Registrations page
- Auction page
- Dedicated Sponsors page
